# aibangmang2.0

## 安装依赖

`npm --registry=https://registry.npm.taobao.org install`

## 工作流

开发模式：`npm start`

生产模式：`npm run build`

## note

js 可用语法：es6 stage-4 + es6 stage-3

## 一些规范

* 脚本不用任何分号
* 异步回调全部使用 promise
* vue `v-on` 和 `v-bind` 使用简写 `:foo` 和 `@click` 表示
